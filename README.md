# Cursor UI Clone

This project is a **UI recreation of the Cursor website**, built using **HTML and CSS only**.  
The goal of this project is to closely replicate the visual design, layout, typography, and interactions of the original Cursor landing page for learning and practice purposes.

##  Recreated Sections

The following sections of the Cursor website UI have been recreated:

- Header / Navigation Bar  
- Hero Section  
- Call-to-Action Buttons (Join / Get Started styles)  
- Feature Sections  
- Footer Layout and Links  

All sections are built with attention to spacing, alignment, and visual hierarchy to closely match the original UI.
